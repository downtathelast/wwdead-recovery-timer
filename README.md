# WWDead Recovery Timer Module
A combined AP tracking userscript for Urban Dead that merges multiple legacy tools into one unified module.
**This script replaces:**
Recovery Timer
AP Countdown
AP Tracking tools

The legacy versions are preserved in /archive for reference but are no longer maintained.

**Features**
Live AP countdown (HH:MM:SS)
AP recovery tracking
Accurate next AP calculation
Character-specific tracking
Unified UI module
Lightweight and non-intrusive
Designed to integrate with existing WWDead UI tools


**Planned:**
AP alarm / notification
Tab display


**Installation**

Install a userscript manager:
Tampermonkey (recommended)
Violentmonkey
Greasemonkey

Install the script:
Click Install


(You can also install directly from the raw file link.)

**Legacy Scripts**
Older standalone scripts have been retired and moved to /archive.
They are:
No longer updated
Kept for reference only
Replaced by this module

Users should migrate to Recovery Timer Module.

This keeps:
One install
One update path
One UI location
